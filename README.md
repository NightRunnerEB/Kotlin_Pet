# Kotlin_Pet

This project is a simple **cinema ticket management system** with a console menu interface.  
When the program starts, the main menu is displayed with the following options:

1. Sell Ticket  
2. Refund Ticket  
3. View Free Seats at a Session  
4. Edit Movie Data  
5. Edit Session Data  
6. Exit the Program
